# 🚀 React JS Project – Calculator & Todo App

## 📘 Overview
This project is a simple and interactive **React application** built using **Vite**. It includes two main features: a Calculator for basic arithmetic operations and a Todo App for managing daily tasks.

The purpose of this project is to strengthen understanding of **React fundamentals**, including components, state management, and project structure.

---

## 🛠️ Technologies Used
- React JS  
- Vite  
- JavaScript (ES6+)  
- HTML5  
- CSS3
